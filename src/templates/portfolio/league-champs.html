League Champs is a simple web app I designed, that offers players of Riot's League of Legends MOBA an up to date and detailed statistical analysis of every one of the game's playable characters.

Every week a timed process on my server triggers hundreds of thousands of HTTP requests to Riot's public RESTful API, at their predefined rate limit, in order to aggregate data for all ranked matches occuring during that week. That data is then parsed by champion and saved to a remote NoSQL Mongo database.

Upon initialization, the client side javascript asynchronously queries this data through a simple RESTful endpoint on my server. The returned data is then parsed, filtered and visualized client side. A dynamic sizing algorithm, responsive media queries, and efficient animation libraries allow this app to be accessed properly on devices of all sizes and orientations.
