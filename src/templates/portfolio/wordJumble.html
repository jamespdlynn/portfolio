Word Jumble is the first game to integrate the Play Plays meta game and avatars.

It is a multiplayer "word-finder" (similar to the popular Hasbro board game: Boggle) where players are given two minutes to find as many as words possible, with longer words and more difficult letters granting them extra points.

Players can choose from multiple game modes:

  • Solo Games: where a player tries to beat their personal high score.

  • Versus Games: where a player competes online against friends or matched opponents over the course of three rounds.

  • Daily Tournaments: where up to a dozen players compete on the same specialized board for large prizes.
The game also features it's own proprietary player rating and tier system, which is used to both match players of similar abilities to each other as well as function as a progression system for more competitive players.

The client apps are built using the Unity game engine and C# (which gets transpiled into native IOS and Android code), and communicate to the 512 Games AWS server stack primarily using RESTFul HTTPs API calls.

Play Pals: Word Jumble currently holds a 4.8 star rating in both the Google Play and Apple App Stores, with over five thousand combined installs to date.
