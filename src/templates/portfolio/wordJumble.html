Word Jumble is the first (and currently only) game to integrate the Play Plays meta game and avatars. It is available for most Android and iOS devices, and currently holds a 4.8 star rating in all major App Stores.

It is a multiplayer "word-finder", where players are given up to two minutes to find as many words as possible from among 16-25 adjacent letters, with longer words and more difficult letters granting them extra points (similar to the popular Hasbro game "Boggle"). Players can also select up to three unique "boosters" prior to the start of each round for an additional edge.

There are three main game modes:

  • Solo Games: where a player tries to beat their personal high score.
  • Versus Games: where a player competes online against a friend or matched opponent over the course of three rounds.
  • Daily Tournaments: where up to a dozen players compete on the same specialized board for large prizes.
The game features it's own proprietary player rating and tier system, which is used to both match players of similar skill levels to each other, as well as function as a progression / leaderboard system for more competitive players.

The client app is built with the Unity game engine in C# (which gets transpiled into separate native IOS and Android codebases at build time), and communicate to the 512 Games AWS server stack primarily using REST API calls over HTTP(s).
