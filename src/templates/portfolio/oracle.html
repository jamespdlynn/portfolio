Oracle's ROI Calculator is an online marketing tool used to demonstrate the potential savings of Oracle's automation platform.

The app is one of multiple single page applications hosted on an in-house NodeJS RackSpace server, each of which are IFramed onto Oracle's hosted web pages at their own behest. I implemented this app's design by using as few flat image files as possible, instead opting for modern CSS3 techniques to draw the unique component shapes and shadows. This allowed for not only a quicker page load, but also a more responsive display that works well at both desktop and tablet resolutions.

In order to convert the dynamic page output into a downloadable PDF, I had a NodeJS endpoint use the client sent POST data to render and save a temporary static HTML page on the server. A separate PhantomJS process is then kicked off to capture this HTML in PDF format. Te result of this capture is returned in binary as the response to the initial POST request using a mixture inter process watchers and callbacks.
